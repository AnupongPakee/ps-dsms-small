# Mini Debt & Saving Management System (ระบบจัดการหนี้และการออมขนาดเล็ก)

## Features:
- Authentication & User Management
- Debt Management (จัดการหนี้)
- Saving Management (จัดการเงินออม)
- Dashboard
- Reports & Insights

## Tech Stack:
- Frontend: Next.js + TailwindCSS + Recharts 
- Backend: FastAPI
- Database: MongoDB
